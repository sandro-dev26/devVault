# DevVault

A modern, light-weight personal project designed to showcase data, insights, and tools around various web development technologies.

> **Note:** This is a personal, non-commercial project built for learning and experimentation purposes.

---

## Features

- **Tech Insights:** Curated data and overviews of modern frontend and web technologies.
- **Responsive UI:** Clean, minimalist dark-mode layout built with Tailwind CSS.
- **Client-Side Routing:** Fast, seamless navigation powered by React Router.

---

## Tech Stack

- **Frontend:** React, TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Build Tool:** Vite
- **Deployment:** Vercel
